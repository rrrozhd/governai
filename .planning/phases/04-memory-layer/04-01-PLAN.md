    1. **governai/memory/auditing.py** — Create AuditingMemoryConnector (per D-13, D-14, D-15, D-16, D-17, D-18, D-19, D-26):

       ```python
       from __future__ import annotations
       from typing import Any
       from governai.audit.emitter import AuditEmitter, emit_event
       from governai.memory.connector import MemoryConnector
       from governai.memory.models import MemoryEntry, MemoryScope
       from governai.models.common import EventType, JSONValue

       class AuditingMemoryConnector:
           """Decorates a MemoryConnector to emit audit events for all operations.

           Per D-13: Wrapping decorator pattern (mirrors RedactingAuditEmitter).
           Per D-18: Uses emit_event() helper for consistent audit emission.
           Per D-19: Does NOT register memory values with SecretRegistry.
           """

           def __init__(
               self,
               inner: MemoryConnector,
               emitter: AuditEmitter,
               *,
               run_id: str,
               thread_id: str | None = None,
               workflow_name: str,
           ) -> None:
               self._inner = inner
               self._emitter = emitter
               self._run_id = run_id
               self._thread_id = thread_id
               self._workflow_name = workflow_name
       ```

       **read()**: Call `self._inner.read(key, scope, target=target)`, then emit MEMORY_READ with `{"key": key, "scope": scope.value, "found": result is not None}`. Return result.

       **write()**: To determine `created` flag (D-16), call `self._inner.read(key, scope, target=target)` first to check existence. Then call `self._inner.write(key, value, scope, target=target)`. Then emit MEMORY_WRITE with `{"key": key, "scope": scope.value, "created": existed is None}`. CRITICAL: payload must NOT contain "value" (D-15).

       **delete()**: Use try/finally pattern per RESEARCH.md Pitfall 5:
       ```python
       async def delete(self, key: str, scope: MemoryScope, *, target: str | None = None) -> None:
           found = True
           try:
               await self._inner.delete(key, scope, target=target)
           except KeyError:
               found = False
               raise
           finally:
               await emit_event(
                   self._emitter,
                   run_id=self._run_id,
                   thread_id=self._thread_id,
                   workflow_name=self._workflow_name,
                   event_type=EventType.MEMORY_DELETE,
                   payload={"key": key, "scope": scope.value, "found": found},
               )
       ```

       **search()**: Call `self._inner.search(query, scope, target=target)`, then emit MEMORY_SEARCH with `{"query": query, "scope": scope.value, "result_count": len(results)}`. Return results.
